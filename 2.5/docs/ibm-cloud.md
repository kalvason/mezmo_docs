## Mezmo for IBM

IBM offers two logging services powered by Mezmo:

- [IBM Log Analysis](https://www.ibm.com/cloud/log-analysis)
- [IBM Cloud Activity Tracker](https://cloud.ibm.com/catalog/services/ibm-cloud-activity-tracker-with-logdna)

Login into your IBM account [here](https://cloud.ibm.com/login) and get started with the [documentation] ([https://cloud.ibm.com/docs/services/Log-Analysis-with-Mezmo?topic=logdna-getting-started](https://cloud.ibm.com/docs/services/Log-Analysis-with-Mezmo?topic=logdna-getting-started)).

You can quickly enable cluster level logs on IBM Cloud Kubernetes Service by following [this guide](https://cloud.ibm.com/docs/services/Log-Analysis-with-Mezmo/tutorials?topic=logdna-kube).
